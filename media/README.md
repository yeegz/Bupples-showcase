# Media

How the public assets for this showcase are organized.

| Folder | Holds | Status |
|--------|-------|--------|
| [`brand/`](brand/) | Logo and the Pip wordmark | ✅ Final |
| [`app/`](app/) | Real in-app captures, status bar removed | ✅ Home · Activity · Session · Settings |
| [`web/`](web/) | The full web app at `bupples.web.app/app` | ✅ Home |
| [`widgets/`](widgets/) | iOS home-screen widget | ✅ Home widget |
| `demo/` | ~30s screen-recorded walkthrough (GIF / MP4) | 🚧 Recording |

All app captures are real screenshots from the shipping build with the iOS status
bar cropped off, then given a consistent rounded-card + soft-shadow treatment. The
Settings capture has the account email redacted.
